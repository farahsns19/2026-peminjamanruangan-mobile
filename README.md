# Sistem Peminjaman Ruangan Kampus - Mobile

Aplikasi mobile untuk Sistem Peminjaman Ruangan Kampus.  
Project ini dibuat sebagai bagian dari tugas Persiapan PDBL 2026, dengan backend menggunakan ASP.NET Core Web API.

---

## Deskripsi

Aplikasi mobile ini digunakan untuk mengakses data peminjaman ruangan kampus melalui REST API backend.

Saat ini project mobile masih tahap awal (inisialisasi), namun sudah disiapkan dengan struktur folder yang setara dengan backend.

---

## Fitur Utama (Planned)

- Menampilkan daftar peminjaman ruangan
- Tambah peminjaman ruangan
- Update status peminjaman
- Riwayat peminjaman

---

## Tech Stack

- Flutter (Dart)
- REST API Integration
- Backend: ASP.NET Core Web API

---

## Cara Menjalankan Project

### 1. Clone Repository

```bash
git clone https://github.com/farahsns19/2026-peminjamanruangan-mobile.git
cd 2026-peminjamanruangan-mobile
```

### 2. Install Dependency
flutter pub get

### 3. Jalankan Aplikasi
flutter run

## Environment Variables

Project ini menggunakan konfigurasi API backend.
